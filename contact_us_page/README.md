# Contact Us Page

A single-page Contact Us website built as part of an assessment Task.

## Technologies Used
- HTML
- CSS
- Bootstrap 
- Formspree
- Git & GitHub

## Features
- Responsive layout using Bootstrap
- Contact form with email functionality

- Deployed on GitHub Pages

## Challenges
- Understanding form submission without backend
- Deploying using GitHub Pages

## Learnings
- Using third-party form services
- Git workflow
- Bootstrap grid system
